#### This project is built on top of the official [MangaDex](https://mangadex.org/) public [API](https://api.mangadex.org/)

# ⚠️ Disclaimer

This project does not host or redistribute content.
All content is fetched directly from the official MangaDex API.

### Features

- Manga search
- Personal library
- Chapter reader
- Multi-language support
- Per-user reading progress
- Volumes and chapters organization
- User authentication

### Frontend

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- NextAuth
- Next-themes
- React-markdown

### Backend

- Next.js API Routes
- Next-intl
- Prisma ORM
- PostgreSQL
